# Project Kickoff Workflow
This checklist ensures every new project starts with the right structure.

- [x] Step 1: Initialize Git and connect to GitHub.
- [x] Step 2: Create the `.antigravity/skills` folder.
- [x] Step 3: Copy the `onboarding-skill.md` into the skills folder.
- [x] Step 4: Ask the user for the project goal.
- [ ] Step 5: Generate a draft README.md based on that goal.
